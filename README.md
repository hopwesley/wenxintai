### 我们的方法论：科学与人文的双重考量

#### 教育发展理论支撑：

- **加德纳的"多元智能理论"**：我们相信，智能是多元的。孩子的优势可能体现在语言、逻辑-数学、空间、人际、自省等多种智能的不同组合上。选科，正是为了扬其长处，而非补其短板。
- **"T型人才"与"π型人才"**：未来社会需要既具备某一领域的深度（"T"的竖），又拥有广博知识面与跨界合作能力（"T"的横）的人才，甚至是拥有两项专业深度的"π型人才"。我们的评估，正是为了识别并培育这种深度与广度的最佳结合点。

#### 心理学理论支撑：

- **霍兰德职业兴趣理论**：通过评估孩子在现实型、研究型、艺术型、社会型、企业型、常规型六大类型的倾向，将其兴趣与未来大学专业和职业环境进行匹配，确保内在驱动与外部选择的和谐统一。

#### 基本算法考量：

我们的智能算法引擎，并非简单的分数累加。它是一个多维度的动态模型，核心考量包括：

- 学科能力现状与潜能：不仅是当前成绩，更包括成绩的走势、在群体中的相对位置，以及所展现出的思维潜力。
- 稳定的兴趣倾向：通过多次、多形式的评估，剥离临时性因素，找到孩子持久的热爱所在。
- 认知风格与人格特质：孩子是偏向抽象思维还是具象感知？是细致严谨还是开拓创新？这决定了不同学科的学习适配度。
- 外部环境匹配度：结合本省高考政策、高校选考要求大数据以及社会长远发展趋势，进行综合研判。

#### 2.关于"专业版"测评模块的说明

- **模块定位**：
    - 基础版（已完成）：聚焦于能力倾向与职业兴趣（如霍兰德代码），解决"能做什么"与"喜欢什么"的基础匹配问题。
    - 专业版（可选）：包含性格测试（如MBTI等）与价值观探索，深入解决"适合什么环境"与"为何而奋斗"的深层动力问题。
- **为何需要专业版**：性格与价值观是影响职业满意度和长期发展稳定性的核心因素。完成专业版测评，将能获得一份整合了"能力-兴趣-性格-价值观"四维度的、更全面、更深刻的个人发展综合报告。

#### 3.关于数据隐私的严正声明

我们承诺，所有收集的个体数据都将经过严格的匿名化与脱敏处理。任何可以识别到具体个人身份的信息（如姓名、身份证号、学校、班级等）均会被分离并加密存储，分析所使用的数据集为无法追溯到具体个人的群体数据。

这些脱敏后的群体数据将仅用于教育研究、模型优化、公益报告等，为教育部门和学校提供决策参考。

我们严格遵守《中华人民共和国个人信息保护法》及相关法律法规，建立完善的数据安全保护体系。
